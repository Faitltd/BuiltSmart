# FAIT Calculators

This directory contains copies of the FAIT calculator projects:

1. **FlipperCalc** - 8-bit Real Estate Analysis Tool for house flippers
2. **Home Depot Scraper** - Tool for scraping product data from Home Depot
3. **OfferShield** - Contractor quote risk analyzer
4. **HomeHealthScore** - Live voice/chat-based home health report
5. **While-You're-Here Calculator** - Handyman task cost estimator
6. **Remodeling Calculator** - Scope-based remodel cost estimator

Each calculator is a standalone project that can be run independently.

## Running the Calculators

Each calculator has its own setup and run instructions. Please refer to the README.md file in each calculator's directory for specific instructions.
